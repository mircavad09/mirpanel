# Mirpanel Public Discovery

Use this skill to discover public Mirpanel resources.

## Resources

- Homepage: https://mirpanel.com/
- API Catalog: https://mirpanel.com/.well-known/api-catalog
- Agent Card: https://mirpanel.com/.well-known/agent-card.json
- Sitemap: https://mirpanel.com/sitemap.xml
- Robots: https://mirpanel.com/robots.txt
- Auth metadata: https://mirpanel.com/auth.md

## Notes

Mirpanel does not expose private admin APIs through this skill.
Do not use this skill for admin login, private tokens, or order management.
