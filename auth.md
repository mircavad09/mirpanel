# Mirpanel Agent Authentication

Mirpanel currently exposes public discovery metadata for agents, but does not provide protected public APIs that require agent OAuth registration.

## Authentication Status

Public agent OAuth registration is not currently available.

## Public Resources

- Homepage: https://mirpanel.com/
- API Catalog: https://mirpanel.com/.well-known/api-catalog
- Protected Resource Metadata: https://mirpanel.com/.well-known/oauth-protected-resource
- OAuth Authorization Server Metadata: https://mirpanel.com/.well-known/oauth-authorization-server
- Agent Card: https://mirpanel.com/.well-known/agent-card.json
- Sitemap: https://mirpanel.com/sitemap.xml
- Robots: https://mirpanel.com/robots.txt

## Agent Registration

Agents do not need to register to access public discovery resources.

Protected API access is not currently offered.

For support, use the contact options available on https://mirpanel.com/.
